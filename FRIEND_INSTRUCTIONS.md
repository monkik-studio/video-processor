# Video Processor - Quick User Guide

App link:

https://video-processor-7ndp.onrender.com

## What This App Does

This app converts videos for social media formats:

- YouTube: 16:9, 1920x1080
- TikTok / Instagram Reels: 9:16, 1080x1920
- Square: 1:1, 1080x1080

It can also add title text, a PNG logo watermark, optional background music, trim videos, mute original audio, and download processed files as MP4 or ZIP.

## Important Note

The app is hosted on Render Free. If it has not been used for a while, the first load may take around 30-60 seconds. Please wait and refresh once if needed.

## How To Use

1. Open the app link:

   https://video-processor-7ndp.onrender.com

2. Upload one or more videos.

   Supported video files:

   - `.mp4`
   - `.mov`

3. Click **Analyze Videos**.

4. Review the video details:

   - duration
   - resolution
   - FPS
   - file size

5. Choose the output format:

   - YouTube
   - TikTok / Reels
   - Square

6. Choose the category:

   - Property
   - Boating
   - Surfing

7. Optional settings:

   - Add trim start and end time, such as `00:05` to `00:30`
   - Add title text
   - Choose title position: top, center, or bottom
   - Upload a PNG logo watermark
   - Choose logo position
   - Upload background music
   - Mute original audio

8. Click **Process Video** or **Process Videos**.

9. Wait for processing to finish.

10. Download the result:

   - Use **Download MP4** for a single file
   - Use **Download ZIP** for batch exports

## YouTube Metadata

After processing, the app automatically creates:

- YouTube title
- description
- keywords

Use the **Copy** buttons to copy them.

## Tips

- For best results, upload clear, high-resolution videos.
- Use short title overlays, such as:
  - `Luxury Waterfront Home`
  - `Sunset Boat Tour`
  - `Morning Surf Session`
- If uploading multiple videos, the same settings apply to all videos.
- Large files may take a while to upload and process.

## If Something Does Not Work

Try these steps:

1. Refresh the page.
2. Wait 30-60 seconds if the app is waking up.
3. Use smaller video files for testing.
4. Make sure the file is MP4 or MOV.
5. Try processing one video first before uploading a batch.
